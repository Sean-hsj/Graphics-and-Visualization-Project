Download the zip file called group_17_assignment7. Then click the betterbetter_finalproject.pde file and then click on the "Run" button on the top left side on the screen. The user will get access to the Home Screen of our game. By clicking the button of "start game", the user will be able to select the character they would like to use in the game. Then they can control the character's movement around the screen using three arrow keys of the keyboard. To restart the game after failure, the user can click the 'restart' button on the upper right corner.
